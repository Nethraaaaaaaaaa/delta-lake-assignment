# Delta Lake MERGE Assignment

## Objective
Perform incremental data processing using Delta Lake using Databricks.

## Technologies Used
- Databricks
- Apache Spark
- PySpark
- Delta Lake

## Steps Performed
- Loaded customer master dataset
- Cleaned null values
- Removed duplicate records
- Created Delta Table
- Loaded incremental dataset
- Performed MERGE operation
- Validated results

## Output
Existing customer records were updated and new customer records were inserted successfully using Delta Lake MERGE.
